Macro Clusters & Trends

The Landscape Blueprint: Where Modern Agents Win or Stall

01. Auth Fragmentation

OAuth2 dominates (68%) in heavy multi-tenant categories (CRM, Productivity). API Key / Static Token patterns (24%) power the developer, infra, and data categories, unlocking frictionless programmatic access.

02. Gating & Pipeline Moats

Over 85% of platforms offer immediate self-serve infrastructure. The remaining 15% are fiercely gated behind enterprise sales funnels (e.g., PitchBook, Otter.ai, Gladly) or slow approval pipelines (Pinterest).

03. The Native MCP Surge

Native AI and data pipelines show rapid deployment of Model Context Protocol (MCP) servers (e.g., Devin, Reducto, Firecrawl), validating Composio's strategic roadmap focus on standard formats.

04. Core Pipeline Blockers

System blockages stem almost entirely from absent developer documentation portals (Pylon, Waterfall.io), changing authentication paths, or legacy web frameworks.

Easy Wins: Developer platforms + productivity suites High Outreach Required: FinTech and heavily siloed B2B CRMs
